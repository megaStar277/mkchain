sslmerge
===================


A tool to help create the correct chain of SSL certificates delivered.

----------


Usage
-------------

```
  Usage:
    sslmerge [option|long-option]

  Examples:
    sslmerge --help
    sslmerge --debug

  Options:
    -h, --help                  show this message
    -d, --debug                 display information on the screen (debug mode)
```
